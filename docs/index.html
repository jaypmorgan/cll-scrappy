Lurvescrappy, a webpage dedicated to the distribution of command line ‘lurves’ mentioned in the UK Ubuntu Podcast (http://ubuntupodcast.org). This site is driven by a very simple, if not crude, python script. If you notice any commands missing, give us a quick email or contribute by helping making the script smarter!
